
# Classroom Management System

Full-stack classroom management platform.

Stack:
- Django + Django REST Framework
- MongoDB
- React + Tailwind
- JWT Authentication
- Chart.js Analytics
- Report Card PDF
- Docker Deployment
- CI/CD with GitHub Actions

Run with Docker:

docker-compose up --build
