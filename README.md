# Classroom Management System

A full-stack classroom management platform built using:

Backend:
- Django
- Django REST Framework
- MongoDB

Frontend:
- React
- TailwindCSS
- Chart.js

Features:
- Student Management
- Attendance Tracking
- Marks and Exams
- Role-based Authentication
- Admin / Teacher / Student Portals
- Analytics Dashboard
- PDF Report Cards

Tech Stack:
Python (Django)
React
MongoDB
JWT Authentication

---

## System Architecture

React Frontend
        ↓
Django REST API
        ↓
MongoDB Database
